## [0.1.0] - 2026-01-30

### 🎉 首次发布 | Initial Release

#### 新增 | Added

- ✨ 音乐心情日记 - 支持粘贴音乐链接记录心情
- 🎵 音乐链接解析 - 自动解析网易云音乐歌曲信息
- 🎨 五色心情系统 - 开心/平静/忧伤/激动/治愈
- 🤖 AI 心情识别 - 根据文字内容智能分析情绪
- 💬 AI 陪聊 - 温暖的 AI 倾听者随时陪伴
- 📊 情绪热力图 - 类似 GitHub 贡献图的年度心情分布
- 📻 心情电台 - 随机播放历史记录中的音乐
- 📈 情绪洞察 - AI 分析情绪模式并给出建议
- 🔐 用户认证 - 基于 Supabase Auth 的邮箱登录
- ☁️ 云端同步 - 数据实时同步，多设备使用
- 🎨 现代 UI - 毛玻璃效果、流畅动画、响应式设计
- 🔌 多 AI 支持 - OpenAI、Claude、智谱、通义千问

#### 技术栈 | Tech Stack

- Next.js 14 (App Router)
- Tailwind CSS
- Framer Motion
- Supabase (PostgreSQL + Auth)
- TypeScript
